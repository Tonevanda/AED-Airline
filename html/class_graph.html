◆ getFlights()

void Graph::getFlights ( string  file)

Reads the flights.csv file and, line by line, checks if there is already a flight from that source Airport to the same target Airport-if there is then it adds the airline of that flight to the list of airlines of that flight, if there isn't then it connects the two airports through the addEdge method. It also keeps count of the number of flights it reads and attributes that value to nFlights in the Graph. Time-complexity -> O(n)
